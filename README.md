## How Styles Work

- Each HTML page has an inline `<style type="text/tailwindcss">` block containing
  the `@theme inline` mapping and `:root` design tokens. This is required by the
  Tailwind v4 browser runtime to resolve utility classes like `bg-background`.
- `styles.css` contains the shared `:root` variable blocks (clustered by theme)
  plus any page-specific non-Tailwind CSS scoped to `[data-page="..."]`.

## How Scripts Work

named IIFEs to prevent variable collisions across pages.
